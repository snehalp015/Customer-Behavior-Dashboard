# 📊 Customer Behavior Dashboard

## 📌 Project Overview

The Customer Behavior Dashboard is an interactive Power BI project designed to analyze customer purchasing behavior and provide meaningful business insights. The dashboard enables users to explore customer demographics, sales performance, revenue trends, and product category analysis using interactive visualizations and filters.

---

## 🎯 Objectives

- Analyze customer purchasing behavior.
- Compare revenue across product categories.
- Analyze sales by category.
- Identify customer demographics.
- Build an interactive dashboard for business decision-making.

## 📂 Dataset

The dataset contains customer shopping information including:

- Review Rating
